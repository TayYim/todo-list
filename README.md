### Filter and Map

In order to finish the quiz, you may need to be familar with Js array `filter` and `map` methods.

```js
const items = [1, 2, 3, 4];
const index = 2;
const filteredItems = items.filter((x, i) => i !== index);
```

```js
const items = [1, 2, 3, 4];
const doubledItems = items.map(x => x * 2); 
```
